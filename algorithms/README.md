## Arbitrary notes

We say that the sqrt(-1) = i or j (Python uses j)
sqrt(-4) = 2i
sqrt(-1 * 4) == sqrt(-1) * sqrt(4)
sqrt(18) == sqrt(9 * 2) == sqrt(9) * sqrt(2)

In Modulo 12, -7 is the additive complement of 5 because they produce the same result.
